1. การเข้าสู่ระบบ (Login)

1
เปิดลิงก์ระบบ: https://recruitmakrocareer.github.io/AEC/
2
กรอก อีเมลของบริษัทคุณ ที่ลงทะเบียนไว้กับ Admin แล้วกด เข้าสู่ระบบ (Login)
3
ระบบจะแสดงชื่อบริษัท (Vendor) ของคุณที่มุมซ้ายบน
เข้าไม่ได้? "อีเมลนี้ไม่มีสิทธิ์เข้าใช้งาน" = อีเมลยังไม่ถูกลงทะเบียน กรุณาติดต่อ Admin (ทีม Recruitment CP Axtra) เพื่อเพิ่มอีเมลของคุณ
💡 คุณจะเห็นเฉพาะข้อมูลของบริษัทตัวเองเท่านั้น — ไม่เห็นข้อมูลของ vendor เจ้าอื่น

2. หน้า "📝 บันทึกข้อมูล" — ผลสัมภาษณ์ & คนเริ่มงาน

ใช้บันทึกยอด ผู้ผ่านสัมภาษณ์ (Interview Passed) และ ผู้เริ่มงาน (Started Work) เป็นรายสาขา แยกตำแหน่ง ล่าม / Bakery / Other

ขั้นตอนบันทึกผลสัมภาษณ์ (ตารางที่ 1)

  1. เลือก วันที่สัมภาษณ์ — ระบบจะดึงข้อมูลเดิมของวันนั้น (ถ้ามี) มาให้แก้ไขได้
  2. กด + เพิ่มสาขา แล้วเลือกสาขาจากรายการ
  3. กรอกจำนวนคนแยกตามตำแหน่ง ล่าม / Bakery / Other
  4. ใส่หมายเหตุได้ที่ช่อง Remark (ถ้ามี)

ขั้นตอนบันทึกคนเริ่มงาน (ตารางที่ 2)

  1. เลือก วันที่เริ่มงาน — ต้องเป็นวันจันทร์เท่านั้น (ระบบตั้งวันจันทร์ถัดไปให้อัตโนมัติ)
  2. เพิ่มสาขาและกรอกจำนวนเช่นเดียวกับตารางแรก

การบันทึก

✓
ตรวจสอบยอดรวมท้ายตาราง แล้วกดปุ่ม 💾 บันทึกข้อมูลทั้งหมด (SAVE & UPDATE DATA) — ปุ่มเดียวบันทึกทั้ง 2 ตาราง
⚠️ สำคัญมาก: การบันทึกเป็นการ "เขียนทับข้อมูลทั้งวัน" ของวันที่เลือก — ข้อมูลเดิมของวันนั้น (เฉพาะของบริษัทคุณ) จะถูกแทนที่ด้วยที่เห็นบนจอทั้งหมด ดังนั้นถ้าจะแก้ไข ให้เลือกวันที่ → รอระบบโหลดข้อมูลเดิมขึ้นมา → แก้ → แล้วค่อยบันทึก
✅ ถ้าบันทึกสำเร็จจะขึ้นข้อความยืนยัน — ถ้าล้มเหลวระบบจะแจ้ง error และข้อมูลที่กรอกไว้จะไม่หาย กดบันทึกซ้ำได้

3. หน้า "📁 อัปโหลดไฟล์ (Onboard)" — ส่งยอดจากไฟล์ Excel

ทางลัดสำหรับส่งยอดคนเริ่มงานทีละหลายสาขาจากไฟล์ Excel โดยไม่ต้องคีย์ทีละแถว

  1. เลือก วันที่เริ่มงาน (วันจันทร์เท่านั้น)
  2. แนบไฟล์ Excel (.xlsx) ที่มีคอลัมน์ เลขที่สาขา และ ตำแหน่ง
  3. กด 🚀 ประมวลผลและส่งยอดคนเริ่มงานทันที

ระบบจะนับจำนวนคนต่อสาขา แยกตำแหน่งให้อัตโนมัติ (ตำแหน่งที่มีคำว่า "ล่าม" → ล่าม, "Bakery/เบเกอรี่" → Bakery, อื่นๆ → Other) และเก็บไฟล์ต้นฉบับไว้ให้ Admin ตรวจสอบ

4. หน้า "📋 Name List" — อัปโหลดรายชื่อพนักงาน

ใช้ส่งรายชื่อพนักงานรายบุคคล พร้อมข้อมูลเอกสาร (passport, work permit ฯลฯ) และติดตามสถานะการเริ่มงานของแต่ละคน

4.1 ดาวน์โหลดฟอร์ม

1
กด ⬇️ ดาวน์โหลดฟอร์มเปล่า — จะได้ไฟล์ Excel (.xlsx) ที่มี dropdown ให้เลือกในช่อง Status, ตำแหน่ง, เพศ, ประเภทผู้สมัคร

4.2 กรอกข้อมูล

  • กรอก 1 คนต่อ 1 แถว — ต้องมีชื่อ-สกุล (ไทยหรืออังกฤษ) ทุกแถว
  • คอลัมน์วันที่ทุกช่อง ใช้รูปแบบ YYYY-MM-DD เช่น 2026-07-13 (ระบบจะจัด format ให้เหมือนกันทุกเจ้าอัตโนมัติ)
  • ช่อง Agency Name เว้นว่างได้ — ระบบใส่ชื่อบริษัทของคุณให้เองตอนอัปโหลด (แก้เป็นชื่ออื่นไม่ได้)

4.3 อัปโหลด

2
เลือกไฟล์ที่กรอกแล้ว → กด 🚀 อัปโหลดรายชื่อ — รายชื่อจะต่อท้ายของเดิม (อัปโหลดเพิ่มได้เรื่อยๆ) และเก็บแยกของแต่ละบริษัท

4.4 อัปเดตสถานะรายคน

แต่ละรายชื่อมี dropdown สถานะ 3 แบบ:

รอเริ่มงาน  เริ่มงาน  ไม่มาเริ่มงาน

  1. เปลี่ยนสถานะในตารางได้เลย (เปลี่ยนได้หลายคนพร้อมกัน — ที่แก้ไว้ไม่หายแม้เปลี่ยนตัวกรอง)
  2. กด 💾 บันทึกสถานะ เพื่อยืนยันทั้งหมดในครั้งเดียว
  3. ใช้ช่อง "กรองสถานะ" เพื่อดูเฉพาะกลุ่มที่ต้องการ เช่น เฉพาะคนที่ "รอเริ่มงาน"
💡 บริษัทของคุณเห็นเฉพาะรายชื่อของตัวเอง — Admin เท่านั้นที่เห็นรายชื่อของทุกเจ้า

5. หน้า "🏢 สาขา & ประวัติ"

  • 🎯 เป้าหมายของบริษัท — Target / Passed / Started / Remaining / % ความคืบหน้าของบริษัทคุณ พร้อมตารางรายสาขาที่คุณรับผิดชอบ
  • 🌍 ภาพรวมทั้งหมด (Grand Total) — ภาพรวมเป้าหมายทุกสาขาของโครงการ
  • ⏱️ ประวัติการอัปเดต — รายการที่บริษัทคุณเคยบันทึก (วันที่ สาขา ตำแหน่ง จำนวน)
💡 ข้อมูลรีเฟรชอัตโนมัติทุก 2 นาที หรือกด 🔄 Refresh Data ในหน้าบันทึกข้อมูลเพื่อดึงทันที

6. กฎสำคัญที่ต้องรู้

เรื่องกฎ
วันเริ่มงานเลือกได้เฉพาะ วันจันทร์ เท่านั้น (ทั้งหน้าบันทึกและหน้าอัปโหลด)
การบันทึกซ้ำวันเดิมเป็นการเขียนทับข้อมูลทั้งวันของบริษัทคุณ — โหลดข้อมูลเดิมขึ้นมาก่อนแก้เสมอ
สาขาที่บันทึกได้ต้องเป็นสาขาที่อยู่ในโครงการเท่านั้น — ถ้าสาขาไหนไม่ถูกบันทึก ระบบจะแจ้งเตือน
ขอบเขตข้อมูลเห็น/แก้ไขได้เฉพาะข้อมูลของบริษัทตัวเอง
รูปแบบวันที่YYYY-MM-DD (เวลาไทย) เหมือนกันทุกบริษัท
Agency Name ใน Name Listระบบใส่ให้อัตโนมัติตามบัญชีที่ login — ปลอมชื่อเจ้าอื่นไม่ได้

7. ปัญหาที่พบบ่อย (FAQ)

Q: เปิดหน้าเว็บแล้ว login ไม่ได้ ขึ้น "เชื่อมต่อ Server ไม่ได้"

ลองเปิดด้วย Google Chrome (Microsoft Edge บางเครื่องบล็อกการเชื่อมต่อ) หรือใช้ลิงก์สำรอง (ขอจาก Admin) — ถ้ายังไม่ได้ให้ตรวจอินเทอร์เน็ตของท่าน

Q: โหลดข้อมูลครั้งแรกช้า

การเข้าครั้งแรกของวันอาจใช้เวลา 3–5 วินาที (ระบบเตรียมข้อมูล) ครั้งถัดไปจะเร็วขึ้น

Q: กดบันทึกแล้วขึ้น error

ข้อมูลที่กรอกไว้ยังอยู่ครบ — อ่านข้อความ error แล้วกดบันทึกซ้ำ ถ้ายังไม่ได้ให้แคปหน้าจอส่งให้ Admin

Q: อัปโหลด Name List แล้วขึ้น "ไม่พบรายชื่อในไฟล์"

ตรวจว่า (1) ใช้ฟอร์มที่ดาวน์โหลดจากระบบ (2) คอลัมน์ชื่อ-สกุล ไม่ว่าง (3) ไม่ได้ลบแถวหัวตารางออก

Q: เลือกวันเริ่มงานแล้วระบบเด้งเตือน

วันเริ่มงานต้องเป็นวันจันทร์เท่านั้น — เลือกวันจันทร์ของสัปดาห์ที่พนักงานเริ่มงานจริง

Q: อยากแก้รายชื่อ/ลบรายชื่อที่อัปโหลดผิด

แจ้ง Admin เพื่อแก้ไขในระบบหลังบ้าน (vendor แก้ได้เฉพาะสถานะ)
